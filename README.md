# TBotLeecher
A twitter crawler to get media from Twitter (in the future I'll release new versions for other services...)

# Commands
* **list**: Lists current contents of the leecher.
* **add_user** *username*: Adds a new username to the leech queue
* **add_hashtag** *#hashtag*: Adds a new hashtag to the leech queue
* **remove** *#hashtag/username*: Removes a username or hashtag from the database
* **remove all**: Cleans the leech queue
* **leech**: self-explanatory
* **daemon**: Enters into daemon mode, and each 250 seconds will attempt to leech
* **set destination** *destinationFolder*: Sets the destination folder of the leeched media
